DO NOT DELETE THIS FOLDER
IT CONTAINS CUSTOMIZED CSS SETABLE BY USER

BY DEFAULT CSS IS SET TO css/default.css

This folder must contain following structure to permit this feature to works :
  /css/custom/<customName>/custom.css  : this file contain replacement rules for standard website css
                          /mobile.css  : use this file to set css for mobile devices
                          /print.css   : this file contain remplacement rules for printing.
